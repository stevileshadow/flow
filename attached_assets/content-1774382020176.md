[Qu'est-ce qui est inclus dans l'abonnement ?](https://www.odoo.com/fr_FR/pricing#)

Tous nos forfaits incluent l’accès à des centaines d’applications Odoo, hébergées sur une infrastructure cloud évolutive,
avec notamment l’hébergement, des sauvegardes quotidiennes incrémentielles sur deux continents, l’intégration d'e-mails
une sécurité de premier ordre, une surveillance 24h/24 et 7j/7, ainsi qu’un centre de contrôle pour gérer votre environnement Odoo.


Notre service inclut un accès illimité à notre support par e-mail, du lundi au vendredi,
24h/24, en anglais, français, espagnol ou néerlandais. L’accès à notre plateforme eLearning est également gratuit.
Les mises à jour sont incluses dans tous nos forfaits, sans frais supplémentaires.


Consultez nos [Conditions générales détaillées](https://www.odoo.com/fr_FR/legal)

Ce qui n'est pas inclus dans ces plans :


- Hébergement sur Odoo.sh pour les développements personnalisés (l'hébergement sur Odoo Online est gratuit, avec un stockage illimité jusqu'à 100 Go [¹](https://www.odoo.com/fr_FR/acceptable-use#abuse))
- Service de mise en œuvre et accès aux services d'un expert Odoo
- Crédits d’achats intégrés : SMS, autocomplétion des contacts, génération de pistes,
IA pour scanner les factures fournisseurs et les notes de frais.

- Maintenance du code personnalisé

[Que signifie On-premise / Odoo.sh?](https://www.odoo.com/fr_FR/pricing#)

Le plan standard est hébergé sur Odoo Online, notre infrastructure cloud pour héberger des bases de données sans modules personnalisés.
Avec le plan personnalisé, vous pouvez héberger sur Odoo Online sans coût supplémentaire ou choisir l’une des deux options suivantes :


1. Téléchargez Odoo Enterprise et occupez-vous vous-même de l'hébergement
2. Hébergez sur Odoo.sh, la plateforme d’intégration continue pour héberger vos branches de développement,
    de préproduction et de production, vous permettant de développer ou d’utiliser des modules personnalisés.


Plus d'informations sur [https://odoo.sh](https://odoo.sh/)

[Où puis-je obtenir des services de mise en œuvre et combien cela coûte-t-il ?](https://www.odoo.com/fr_FR/pricing#)

Si vous avez besoin de services de mise en œuvre : gestion de projet, analyse des besoins métiers,
importation de données, personnalisations, développements, formations, etc., nous proposons deux options :


1. Les petites entreprises (<50 employés) travaillent généralement directement avec Odoo,
    et utilisent [Success Packs](https://www.odoo.com/fr_FR/pricing-packs).
    Découvrez notre estimateur de projet pour connaître le coût d’un projet en fonction de vos besoins.

2. Les entreprises de taille moyenne ou de grande taille (plus de 50 employés) font généralement appel à un partenaire
    offrant des services de gestion de projet localisés :
    [https://odoo.com/partners](https://www.odoo.com/fr_FR/partners)
3. Pour les grandes entreprises (+250 employés), contactez Odoo directement :
    [https://odoo.com/contactus](https://www.odoo.com/fr_FR/contactus)

[Est-ce que l'environnement à plusieurs entreprises ou Studio est disponible dans le plan Une App Gratuite ?](https://www.odoo.com/fr_FR/pricing#)

Oui, le plan gratuit permet de gérer plusieurs sociétés.

Vous pouvez également choisir Odoo Studio comme application pour le plan gratuit.


[Pourquoi ai-je plusieurs applications avec le plan Une App Gratuite ?](https://www.odoo.com/fr_FR/pricing#)

Si vous choisissez une application qui dépend d'autres applications (par exemple, eCommerce dépend de Site Web et
Facturation), ces applications requises vous sont également offertes. Vous pouvez donc commencer avec n'importe quelle application
et nous garantissons que son utilisation est gratuite pour un nombre illimité d’utilisateurs, pour toujours, quelles que soient les
dépendances de l’application.


[Comment migrer d'un plan Une App Gratuite à un plan Standard ou Personnalisé ?](https://www.odoo.com/fr_FR/pricing#)

Pour passer du plan Une App Gratuite au plan Standard, il vous suffit d’installer les applications
supplémentaires dont vous avez besoin. Si vous installez Studio ou ajoutez des sociétés à votre base de données,
vous passerez automatiquement au plan Personnalisé.


[Quelle est la différence entre le plan standard et le plan personnalisé ?](https://www.odoo.com/fr_FR/pricing#)

Le plan personnalisé s’adresse aux entreprises qui souhaitent gérer plusieurs sociétés sur une seule
base de données et personnaliser Odoo via Odoo Studio,
des développements sur mesure ou l’API.


[Comment définissez-vous un utilisateur payant ?](https://www.odoo.com/fr_FR/pricing#)

Un utilisateur payant est généralement un employé ayant accès au backend d’Odoo pour créer,
consulter ou modifier des documents.
Les utilisateurs externes (tels que les clients ou les fournisseurs) accédant à Odoo via le portail ne sont pas considérés comme
des utilisateurs payants.


Exemples d'utilisateurs payants :


- Un employé qui enregistre ses notes de frais ou sa feuille de temps
- Un comptable externe qui utilise Odoo pour travailler sur vos finances

Exemples d'utilisateurs gratuits :


- Les clients et les fournisseurs qui ont accès à leurs factures sur le portail
- Les visiteurs de votre site web qui placent des commandes eCommerce ou qui suivent leurs livraisons

[Puis-je passer d'un plan hébergé (Odoo Online) à Odoo Enterprise ou inversement ?](https://www.odoo.com/fr_FR/pricing#)

Lorsque vous utilisez Odoo Online, vous pouvez télécharger une sauvegarde de votre base de données à tout moment via
le centre de contrôle. Vos données vous appartiennent !


Odoo Online ou Odoo Enterprise (On-premise ou Odoo.sh) utilisent le même logiciel. Seuls
l’hébergement et l’infrastructure diffèrent : e-mails, sauvegardes, redondance des bases de données, etc.


Notre infrastructure cloud fonctionne avec la dernière version d’Odoo et bénéficie de mises à jour régulières.
Les utilisateurs on-premise doivent utiliser la dernière version stable d’Odoo.
Si vous souhaitez passer d’Odoo Online à Odoo On-premise,
il est préférable de le faire immédiatement après la sortie de la dernière
version stable.


[Que signifie API externe ?](https://www.odoo.com/fr_FR/pricing#)

API externe signifie que vous configurez un logiciel externe (tel que des systèmes existants, des moteurs de rapports, etc.) qui lance des appels à l'API Odoo (JSON-RPC ou XML-RPC) pour interroger, récupérer ou mettre à jour les données stockées dans Odoo.


Lorsque l'appel est lancé depuis Odoo, ce n'est PAS considéré comme une "API externe". Par exemple, lorsqu'une application Odoo utilise un service externe tel qu'un fournisseur de paiement (Stripe, PayPal, Adyen, etc.), un transporteur (UPS, FedEX, DHL, etc.), un fournisseur de VoIP ou la synchronisation bancaire, cela fait partie du plan standard.
